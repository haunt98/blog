# [Go](https://golang.org/)

| Distribution | Package          |
| ------------ | ---------------- |
| Arch Linux   | `go`, `go-tools` |

## Configuration

Add to `~/.bashrc`, `~/.zshrc`:

```sh
export GOPATH=$HOME/go
export PATH="$PATH:$GOPATH/bin"
```
