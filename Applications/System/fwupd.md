# [fwupd](https://github.com/fwupd/fwupd)

| Distribution | Package                             |
| ------------ | ----------------------------------- |
| Arch Linux   | `fwupd`                             |
| Ubuntu       | `fwupd`                             |
| Fedora       | `fwupd`                             |
| Snap         | [fwupd](https://snapcraft.io/fwupd) |

Usage flow:

```
fwupdmgr get-devices

fwupdmgr refresh

fwupdmgr get-updates

fwupdmgr update
```
