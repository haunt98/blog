# [bat](https://github.com/sharkdp/bat)

| Distribution | Package |
| ------------ | ------- |
| Arch Linux   | `bat`   |
| Ubuntu       | `bat`   |

List themes:

```sh
bat --list-themes
```

Edit config file at `~/.config/bat/config`:

```txt
--theme="YourFavoriteTheme"

--style="plain"
```

See more config option:

```sh
bat --help
```
