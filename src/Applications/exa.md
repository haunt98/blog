# [exa](https://github.com/ogham/exa)

| Distribution | Package |
| ------------ | ------- |
| Arch Linux   | `exa`   |
| Ubuntu       | `exa`   |
| Homebrew     | `exa`   |

Add to `~/.bashrc`:

```bash
# Arch Linux, Ubuntu
[[ -f /usr/bin/exa ]] && \
    alias ls="exa"

# Homebrew
[[ -f /usr/local/bin/exa ]] && \
    alias ls="exa"
```
